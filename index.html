Airwave

A macOS application that provides system-wide spatial audio through headphones using HRIR convolution. Built as a free, customizable alternative to proprietary spatial audio solutions — because tools like these shouldn't cost a subscription.

Routes all system audio through a real-time convolution pipeline at ~0.1% CPU. Compatible with HeSuVi HRIR presets. Written in Swift.

is-framework

Canonical monorepo framework for South Neuhof information system frontends. Standardized on pnpm workspaces, Vite, and Turbo, it provides the core infrastructure for scalable Vue 3 and Expo mobile client applications.

Publishes shared model schemas, API contracts, and components under the @southneuhof scope. Enables source-first development with Vite HMR mappings and strict package boundary checks.

landing-sveltekit

Reusable SvelteKit landing page engine primitives for South Neuhof applications. Built specifically to deliver high-performance, modular, and aesthetic marketing layers for the ecosystem.

Features optimized layout grids, pre-configured SEO-friendly routing, and strict core vitals compliance (100% Mobile/Desktop LCP). Mirrored and published directly from the core information system monorepo.

apostle

A type-safe HTTP client wrapper and request lifecycle orchestration package. It serves as the unified network communication layer for South Neuhof frontends, published under the @southneuhof scope.

Provides automated retry algorithms, global authorization header injection, and response payload validation helper utilities. Engineered to maintain low overhead on mobile and browser targets.
